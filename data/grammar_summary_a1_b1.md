## 1. Nomen & Artikel

### 1.1 Drei Geschlechter (Genus)

Im Deutschen hat jedes Nomen ein grammatisches Geschlecht. Man muss das Geschlecht mit dem Nomen zusammen lernen.

| Genus    | Artikel | Beispiele                      |
| -------- | ------- | ------------------------------ |
| Maskulin | der     | der Mann, der Tisch, der Baum  |
| Feminin  | die     | die Frau, die Lampe, die Stadt |
| Neutrum  | das     | das Kind, das Buch, das Haus   |

**Tipps zur Erkennung:**

- Maskulin: Tage (der Montag), Monate (der Januar), Jahreszeiten (der Sommer), männliche Personen (der Vater)
- Feminin: Endungen -ung (die Zeitung), -heit (die Freiheit), -keit (die Möglichkeit), -tion (die Nation)
- Neutrum: Endungen -chen (das Mädchen), -lein (das Büchlein), Metalle (das Gold), Infinitive als Nomen (das Essen)

---

### 1.2 Pluralformen

Es gibt fünf Pluralmuster im Deutschen. Es gibt keine feste Regel — man muss den Plural mitlernen.

| Typ                           | Singular → Plural       | Beispiele              |
| ----------------------------- | ----------------------- | ---------------------- |
| -e                            | der Tisch → die Tische  | der Hund → die Hunde   |
| -er (oft + Umlaut)            | das Kind → die Kinder   | das Haus → die Häuser  |
| -en / -n                      | die Frau → die Frauen   | die Lampe → die Lampen |
| -s                            | das Auto → die Autos    | das Foto → die Fotos   |
| – (kein Suffix, evtl. Umlaut) | der Lehrer → die Lehrer | der Vater → die Väter  |

**Beispielsätze:**

- _Die Kinder spielen im Garten._
- _Ich habe zwei Autos gesehen._
- _Die Häuser in dieser Straße sind alt._

---

### 1.3 Vier Fälle (Kasus)

Jedes Nomen steht in einem der vier Fälle, je nach seiner Funktion im Satz.

| Fall      | Frage     | Funktion          | Beispiel                         |
| --------- | --------- | ----------------- | -------------------------------- |
| Nominativ | Wer? Was? | Subjekt           | **Der Mann** liest ein Buch.     |
| Akkusativ | Wen? Was? | Direktes Objekt   | Ich sehe **den Mann**.           |
| Dativ     | Wem?      | Indirektes Objekt | Ich gebe **dem Mann** das Buch.  |
| Genitiv   | Wessen?   | Besitz            | Das Auto **des Mannes** ist rot. |

---

### 1.4 Artikeldeklination

Die Artikel ändern sich je nach Genus, Numerus und Kasus.

**Bestimmter Artikel (der, die, das):**

| Fall      | Maskulin    | Feminin | Neutrum     | Plural   |
| --------- | ----------- | ------- | ----------- | -------- |
| Nominativ | der         | die     | das         | die      |
| Akkusativ | den         | die     | das         | die      |
| Dativ     | dem         | der     | dem         | den (+n) |
| Genitiv   | des (+s/es) | der     | des (+s/es) | der      |

**Unbestimmter Artikel (ein, eine, ein):**

| Fall      | Maskulin      | Feminin | Neutrum       |
| --------- | ------------- | ------- | ------------- |
| Nominativ | ein           | eine    | ein           |
| Akkusativ | einen         | eine    | ein           |
| Dativ     | einem         | einer   | einem         |
| Genitiv   | eines (+s/es) | einer   | eines (+s/es) |

**Beispielsätze:**

- _Ich sehe **den** Hund._ (Akkusativ, maskulin)
- _Sie gibt **einem** Kind das Spielzeug._ (Dativ, neutrum)
- _Das ist das Auto **der** Frau._ (Genitiv, feminin)
- _Er hilft **den** Kindern._ (Dativ, Plural — Nomen bekommt -n)

---

### 1.5 N-Deklination (schwache Maskulina)

Einige maskuline Nomen bekommen in allen Fällen außer Nominativ die Endung **-(e)n**. Diese Nomen nennt man „schwache Maskulina".

**Typische N-Deklination-Nomen:**

- Männliche Personen auf -e: der Junge, der Kollege, der Kunde, der Türke
- Nationalitäten auf -e: der Franzose, der Grieche, der Pole
- Einige ohne -e: der Herr, der Mensch, der Nachbar, der Student, der Polizist

| Fall      | Singular          | Plural        |
| --------- | ----------------- | ------------- |
| Nominativ | der Student       | die Studenten |
| Akkusativ | den Student**en** | die Studenten |
| Dativ     | dem Student**en** | den Studenten |
| Genitiv   | des Student**en** | der Studenten |

**Beispielsätze:**

- _Ich frage **den Kollegen**._ (nicht: den Kollege)
- _Er hilft **dem Jungen**._ (nicht: dem Junge)
- _Das ist das Büro **des Herrn** Müller._ (Herr → Herrn)
- _Kennst du **den Nachbarn**?_

## 2. Verben & Konjugation

### 2.1 Präsens (Gegenwart)

Das Präsens beschreibt aktuelle Handlungen, Gewohnheiten und allgemeine Wahrheiten.

**Regelmäßige Verben (z.B. _machen_):**

| Person    | Endung | Beispiel   |
| --------- | ------ | ---------- |
| ich       | -e     | ich mache  |
| du        | -st    | du machst  |
| er/sie/es | -t     | er macht   |
| wir       | -en    | wir machen |
| ihr       | -t     | ihr macht  |
| sie/Sie   | -en    | sie machen |

**Unregelmäßige Verben — Vokalwechsel:**

| Verb           | ich     | du       | er/sie/es |
| -------------- | ------- | -------- | --------- |
| fahren (a→ä)   | fahre   | fährst   | fährt     |
| sprechen (e→i) | spreche | sprichst | spricht   |
| lesen (e→ie)   | lese    | liest    | liest     |
| nehmen (e→i)   | nehme   | nimmst   | nimmt     |

**Beispielsätze:**

- _Ich **lerne** jeden Tag Deutsch._
- _Er **spricht** drei Sprachen._
- _Wir **fahren** morgen nach Berlin._

---

### 2.2 Perfekt (Vergangenheit)

Das Perfekt ist die häufigste Vergangenheitsform in der gesprochenen Sprache. Bildung: **haben/sein + Partizip II**.

**Partizip II bilden:**

- Regelmäßig: ge- + Stamm + -t → gemacht, gelernt, gekauft
- Unregelmäßig: ge- + Stamm (verändert) + -en → geschrieben, gefahren, gegessen
- Trennbare Verben: Präfix + ge + Stamm → aufgemacht, eingekauft, angefangen
- Untrennbare Verben (be-, emp-, ent-, er-, ge-, miss-, ver-, zer-): kein ge- → besucht, verstanden, erzählt
- Verben auf -ieren: kein ge- → studiert, telefoniert

**Wann „sein" statt „haben"?**

- Bewegung von A nach B: _Ich **bin** nach Hause gegangen._
- Zustandsveränderung: _Er **ist** eingeschlafen._
- Ausnahmen: sein, bleiben, werden → _Ich **bin** zu Hause geblieben._

**Beispielsätze:**

- _Ich **habe** gestern ein Buch **gelesen**._
- _Sie **ist** nach Paris **geflogen**._
- _Wir **haben** den Film **gesehen**._
- _Er **hat** den Kuchen **mitgebracht**._

---

### 2.3 Präteritum (Schriftliche Vergangenheit)

Das Präteritum verwendet man vor allem in der Schriftsprache und bei _sein_, _haben_ und den Modalverben auch in der Alltagssprache.

**sein / haben:**

| Person    | sein  | haben   |
| --------- | ----- | ------- |
| ich       | war   | hatte   |
| du        | warst | hattest |
| er/sie/es | war   | hatte   |
| wir       | waren | hatten  |
| ihr       | wart  | hattet  |
| sie/Sie   | waren | hatten  |

**Modalverben im Präteritum:**

| Verb   | ich/er | wir/sie |
| ------ | ------ | ------- |
| können | konnte | konnten |
| müssen | musste | mussten |
| wollen | wollte | wollten |
| sollen | sollte | sollten |
| dürfen | durfte | durften |

**Beispielsätze:**

- _Ich **war** gestern krank._
- _Wir **hatten** keine Zeit._
- _Er **konnte** nicht kommen._
- _Sie **wollte** das Buch lesen._

---

### 2.4 Plusquamperfekt (Vorvergangenheit) — B1

Das Plusquamperfekt beschreibt eine Handlung, die **vor** einer anderen Handlung in der Vergangenheit passiert ist. Bildung: **hatte/war + Partizip II**.

Es wird oft mit _nachdem_ verwendet.

**Beispielsätze:**

- **\*Nachdem** ich gegessen **hatte**, ging ich spazieren.\*
- **\*Nachdem** sie angekommen **war**, rief sie mich an.\*
- _Er war müde, weil er die ganze Nacht gearbeitet **hatte**._
- **\*Nachdem** wir den Film gesehen **hatten**, gingen wir nach Hause.\*

---

### 2.5 Futur I (Zukunft)

Bildung: **werden + Infinitiv**. Drückt Zukunft, Vermutungen oder Absichten aus.

| Person    | werden |
| --------- | ------ |
| ich       | werde  |
| du        | wirst  |
| er/sie/es | wird   |
| wir       | werden |
| ihr       | werdet |
| sie/Sie   | werden |

**Beispielsätze:**

- _Ich **werde** morgen nach München **fahren**._
- _Es **wird** bestimmt regnen._ (Vermutung)
- _Wir **werden** das Problem **lösen**._

> **Hinweis:** Im Alltag benutzt man oft Präsens + Zeitangabe für die Zukunft: _Ich fahre morgen nach München._

---

### 2.6 Modalverben

Modalverben verändern die Bedeutung des Hauptverbs. Das Hauptverb steht im Infinitiv am Satzende.

| Modalverb | Bedeutung               | Beispiel                            |
| --------- | ----------------------- | ----------------------------------- |
| können    | Fähigkeit / Möglichkeit | Ich **kann** gut schwimmen.         |
| müssen    | Pflicht / Notwendigkeit | Du **musst** heute arbeiten.        |
| wollen    | Wunsch / Absicht        | Wir **wollen** ins Kino gehen.      |
| sollen    | Empfehlung / Auftrag    | Du **sollst** mehr Wasser trinken.  |
| dürfen    | Erlaubnis               | Hier **darf** man nicht rauchen.    |
| mögen     | Vorliebe / Geschmack    | Ich **mag** Schokolade.             |
| möchte    | Wunsch (höflich)        | Ich **möchte** einen Kaffee, bitte. |

**Satzstruktur:**

- _Ich **kann** Deutsch **sprechen**._ (Modalverb in Position 2, Infinitiv am Ende)

---

### 2.7 Trennbare & untrennbare Verben

**Trennbare Verben** haben ein betontes Präfix, das im Hauptsatz vom Verb getrennt wird:

- Präfixe: ab-, an-, auf-, aus-, ein-, mit-, vor-, zu-, zurück-...

| Infinitiv | Hauptsatz                           | Perfekt               |
| --------- | ----------------------------------- | --------------------- |
| aufstehen | Ich **stehe** um 7 Uhr **auf**.     | Ich bin aufgestanden. |
| einkaufen | Er **kauft** im Supermarkt **ein**. | Er hat eingekauft.    |
| anfangen  | Wir **fangen** jetzt **an**.        | Wir haben angefangen. |
| mitkommen | **Kommst** du **mit**?              | Bist du mitgekommen?  |

**Untrennbare Verben** haben ein unbetontes Präfix, das immer am Verb bleibt:

- Präfixe: be-, er-, ver-, ent-, emp-, ge-, zer-, miss-

| Infinitiv | Hauptsatz                        | Perfekt (kein ge-!) |
| --------- | -------------------------------- | ------------------- |
| besuchen  | Ich **besuche** meine Oma.       | Ich habe besucht.   |
| verstehen | Er **versteht** die Frage.       | Er hat verstanden.  |
| erzählen  | Sie **erzählt** eine Geschichte. | Sie hat erzählt.    |

---

### 2.8 Reflexive Verben

Reflexive Verben brauchen ein Reflexivpronomen. Die Handlung richtet sich auf das Subjekt zurück.

**Reflexivpronomen:**

| Person    | Akkusativ | Dativ |
| --------- | --------- | ----- |
| ich       | mich      | mir   |
| du        | dich      | dir   |
| er/sie/es | sich      | sich  |
| wir       | uns       | uns   |
| ihr       | euch      | euch  |
| sie/Sie   | sich      | sich  |

**Beispielsätze:**

- _Ich wasche **mich**._ (Akkusativ)
- _Ich wasche **mir** die Hände._ (Dativ — weil es ein weiteres Akkusativobjekt gibt)
- _Er freut **sich** auf den Urlaub._
- _Wir treffen **uns** um 8 Uhr._
- _Bitte setzen Sie **sich**._

---

### 2.9 Passiv (Vorgangspassiv) — B1

Das Passiv betont die Handlung, nicht die Person. Bildung: **werden + Partizip II**.

| Zeitform   | Bildung                    | Beispiel                        |
| ---------- | -------------------------- | ------------------------------- |
| Präsens    | wird + Partizip II         | Das Haus **wird gebaut**.       |
| Präteritum | wurde + Partizip II        | Das Haus **wurde gebaut**.      |
| Perfekt    | ist + Partizip II + worden | Das Haus **ist gebaut worden**. |

**Beispielsätze:**

- _Aktiv: Man repariert das Auto._ → _Passiv: Das Auto **wird repariert**._
- _Aktiv: Der Chef hat den Brief geschrieben._ → _Passiv: Der Brief **ist** vom Chef **geschrieben worden**._
- _Hier **wird** Deutsch **gesprochen**._
- _Die E-Mail **wurde** gestern **geschickt**._

---

### 2.10 Konjunktiv II — B1

Der Konjunktiv II drückt irreale Wünsche, höfliche Bitten und hypothetische Situationen aus.

**Häufige Formen:**

| Verb   | Konjunktiv II                               |
| ------ | ------------------------------------------- |
| sein   | ich wäre, du wärst, er wäre, wir wären      |
| haben  | ich hätte, du hättest, er hätte, wir hätten |
| können | ich könnte, du könntest, er könnte          |
| werden | ich würde, du würdest, er würde             |

**Bildung für andere Verben:** würde + Infinitiv

- _Ich **würde** gern nach Japan **reisen**._

**Verwendung:**

1. **Höfliche Bitten:**

   - _Könnten Sie mir bitte helfen?_
   - _Hätten Sie einen Moment Zeit?_
   - _Würden Sie das bitte wiederholen?_

2. **Irreale Wünsche:**

   - _Wenn ich reich wäre, würde ich eine Weltreise machen._
   - _Ich hätte gern mehr Freizeit._

3. **Ratschläge:**
   - _An deiner Stelle würde ich zum Arzt gehen._
   - _Du solltest mehr schlafen._

---

### 2.11 Imperativ (Befehle & Aufforderungen)

Der Imperativ drückt Befehle, Bitten und Aufforderungen aus. Es gibt drei Formen: du, ihr und Sie.

**Bildung:**

| Form | Regel               | Beispiel (machen) | Beispiel (lesen) |
| ---- | ------------------- | ----------------- | ---------------- |
| du   | Stamm (+e optional) | Mach(e)!          | Lies!            |
| ihr  | Stamm + t           | Macht!            | Lest!            |
| Sie  | Infinitiv + Sie     | Machen Sie!       | Lesen Sie!       |

**Besonderheiten:**

- Vokalwechsel e→i/ie gilt bei du: _lesen → Lies!_, _sprechen → Sprich!_, _nehmen → Nimm!_
- Vokalwechsel a→ä gilt **nicht** im Imperativ: _fahren → Fahr!_ (nicht: Fähr!)
- „sein" ist unregelmäßig: _Sei ruhig! / Seid pünktlich! / Seien Sie vorsichtig!_

**Trennbare Verben:**

- _Steh auf!_ / _Steht auf!_ / _Stehen Sie auf!_
- _Mach die Tür zu!_

**Beispielsätze:**

- **\*Komm** bitte her!\* (du)
- **\*Öffnet** eure Bücher auf Seite 10!\* (ihr)
- **\*Nehmen Sie** bitte Platz!\* (Sie)
- **\*Sei** nicht so laut!\* (du)
- **\*Vergiss** deinen Schlüssel nicht!\* (du)

## 3. Satzbau & Konnektoren

### 3.1 Hauptsatz — Verb auf Position 2 (V2-Regel)

Im deutschen Hauptsatz steht das konjugierte Verb **immer** auf Position 2.

| Position 1 | Position 2 (Verb) | Mittelfeld      | Ende     |
| ---------- | ----------------- | --------------- | -------- |
| Ich        | lerne             | jeden Tag       | Deutsch. |
| Jeden Tag  | lerne             | ich             | Deutsch. |
| Morgen     | fahre             | ich nach Berlin | —        |

**Wichtig:** Position 1 kann verschiedene Elemente enthalten (Subjekt, Zeitangabe, Objekt...), aber das Verb bleibt auf Position 2.

**Beispielsätze:**

- _Gestern **hat** er mich angerufen._
- _Leider **kann** ich nicht kommen._
- _In der Schule **lernen** die Kinder Deutsch._

---

### 3.2 Fragesätze

**Ja/Nein-Fragen:** Verb auf Position 1

- **\*Lernst** du Deutsch?\*
- **\*Hast** du Zeit?\*
- **\*Können** Sie mir helfen?\*

**W-Fragen:** W-Wort + Verb auf Position 2

- _Was **machst** du?_
- _Woher **kommst** du?_
- _Wann **fährt** der Zug?_
- _Warum **bist** du so müde?_

---

### 3.3 Koordinierende Konjunktionen (Hauptsatz + Hauptsatz)

Diese Konjunktionen verbinden zwei Hauptsätze. Die Wortstellung ändert sich **nicht** — das Verb bleibt auf Position 2 im zweiten Satz.

| Konjunktion | Bedeutung                 | Beispiel                                           |
| ----------- | ------------------------- | -------------------------------------------------- |
| und         | Addition                  | Ich lerne Deutsch **und** ich arbeite bei Siemens. |
| aber        | Gegensatz                 | Ich möchte kommen, **aber** ich habe keine Zeit.   |
| oder        | Alternative               | Möchtest du Tee **oder** trinkst du lieber Kaffee? |
| denn        | Grund                     | Ich bleibe zu Hause, **denn** ich bin krank.       |
| sondern     | Korrektur (nach Negation) | Er trinkt nicht Kaffee, **sondern** er trinkt Tee. |

---

### 3.4 Subordinierende Konjunktionen (Nebensatz — Verb am Ende)

Im Nebensatz steht das konjugierte Verb am **Ende** des Satzes.

| Konjunktion | Bedeutung                             | Beispiel                                                          |
| ----------- | ------------------------------------- | ----------------------------------------------------------------- |
| weil        | Grund                                 | Ich lerne Deutsch, **weil** ich in Deutschland arbeiten **will**. |
| dass        | Inhalt                                | Ich weiß, **dass** er morgen **kommt**.                           |
| wenn        | Bedingung / wiederholte Vergangenheit | **Wenn** es regnet, bleibe ich zu Hause.                          |
| obwohl      | Gegensatz                             | Er geht joggen, **obwohl** es kalt **ist**.                       |
| ob          | Indirekte Ja/Nein-Frage               | Ich weiß nicht, **ob** er Zeit **hat**.                           |
| damit       | Zweck (anderes Subjekt)               | Ich erkläre es langsam, **damit** du es **verstehst**.            |
| bevor       | Zeitlich: vorher                      | **Bevor** ich schlafe, lese ich ein Buch.                         |
| nachdem     | Zeitlich: nachher                     | **Nachdem** er gegessen hatte, ging er spazieren.                 |
| während     | Gleichzeitigkeit                      | **Während** ich koche, hört er Musik.                             |
| seitdem     | Seit einem Zeitpunkt                  | **Seitdem** ich in Berlin wohne, spreche ich besser Deutsch.      |
| bis         | Bis zu einem Zeitpunkt                | Warte, **bis** ich fertig **bin**.                                |
| sobald      | Sofort wenn                           | **Sobald** ich ankomme, rufe ich dich an.                         |

**Wortstellung im Nebensatz:**

```
Konjunktion + Subjekt + ... + konjugiertes Verb (am Ende)
```

- _Ich weiß, **dass** er heute nicht kommen **kann**._ (Modalverb am Ende)
- _..., **weil** ich gestern zu spät ins Bett gegangen **bin**._ (Hilfsverb am Ende)

**Nebensatz auf Position 1 (Verb des Hauptsatzes folgt direkt):**

Wenn der Nebensatz vor dem Hauptsatz steht, zählt der ganze Nebensatz als Position 1:

- **\*Weil** ich müde **bin**, **gehe** ich ins Bett.\* (Verb des Hauptsatzes direkt nach dem Komma)
- **\*Wenn** es regnet, **bleibe** ich zu Hause.\*
- **\*Obwohl** er krank **war**, **ging** er zur Arbeit.\*

---

### 3.5 Satzadverbien / Konnektive Adverbien (mit Inversion)

Satzadverbien stehen auf Position 1 und das Verb bleibt auf Position 2 — **aber das Subjekt rutscht hinter das Verb** (Inversion).

| Adverb                     | Bedeutung       | Beispiel                                                                        |
| -------------------------- | --------------- | ------------------------------------------------------------------------------- |
| deshalb / deswegen / daher | Folge / Grund   | Ich bin müde, **deshalb** bleibe **ich** zu Hause.                              |
| trotzdem                   | Gegensatz       | Es regnet, **trotzdem** gehe **ich** spazieren.                                 |
| außerdem                   | Ergänzung       | Er spricht Deutsch. **Außerdem** spricht **er** Französisch.                    |
| dann / danach              | Zeitliche Folge | Zuerst esse ich. **Dann** gehe **ich** zur Arbeit.                              |
| allerdings                 | Einschränkung   | Das Essen war gut, **allerdings** war **es** zu teuer.                          |
| zuerst / schließlich       | Reihenfolge     | **Zuerst** lernte **ich** Englisch, **schließlich** begann **ich** mit Deutsch. |

**Vergleich der drei Konnektortypen:**

| Typ                             | Wortstellung                    | Beispiel                                           |
| ------------------------------- | ------------------------------- | -------------------------------------------------- |
| Koordinierend (und, aber…)      | Keine Änderung                  | Ich bin müde, **aber** **ich** gehe trotzdem.      |
| Subordinierend (weil, dass…)    | Verb am Ende                    | Ich gehe, **weil** ich müde **bin**.               |
| Satzadverb (deshalb, trotzdem…) | Inversion (V2 + Subjekt danach) | Ich bin müde, **deshalb** gehe **ich** nach Hause. |

---

### 3.6 als vs. wenn

Beide bedeuten „when", aber:

|          | als                                              | wenn                                      |
| -------- | ------------------------------------------------ | ----------------------------------------- |
| Wann?    | Einmalig in der Vergangenheit                    | Wiederholt ODER Gegenwart/Zukunft         |
| Beispiel | **Als** ich ein Kind war, wohnte ich in München. | **Wenn** ich müde bin, trinke ich Kaffee. |
| Beispiel | **Als** ich gestern ankam, regnete es.           | Immer **wenn** ich ihn sah, lachte er.    |

**Faustregel:** Einmal in der Vergangenheit → _als_. Alles andere → _wenn_.

---

### 3.7 Relativsätze — B1

Relativsätze geben zusätzliche Informationen über ein Nomen. Das Relativpronomen richtet sich nach dem Genus des Bezugswortes und dem Kasus im Nebensatz.

**Relativpronomen:**

| Fall      | Maskulin | Feminin | Neutrum | Plural |
| --------- | -------- | ------- | ------- | ------ |
| Nominativ | der      | die     | das     | die    |
| Akkusativ | den      | die     | das     | die    |
| Dativ     | dem      | der     | dem     | denen  |
| Genitiv   | dessen   | deren   | dessen  | deren  |

**Beispielsätze:**

- _Der Mann, **der** dort steht, ist mein Lehrer._ (Nominativ, maskulin)
- _Die Frau, **die** ich gestern getroffen habe, ist Ärztin._ (Akkusativ, feminin)
- _Das Buch, **das** auf dem Tisch liegt, gehört mir._ (Nominativ, neutrum)
- _Die Kinder, **denen** ich geholfen habe, waren sehr nett._ (Dativ, Plural)
- _Der Film, **den** wir gesehen haben, war langweilig._ (Akkusativ, maskulin)
- _Der Mann, **dessen** Auto kaputt ist, nimmt den Bus._ (Genitiv, maskulin)
- _Die Frau, **deren** Kinder hier spielen, wohnt nebenan._ (Genitiv, feminin)

> **Genitiv-Relativpronomen (B1+):** _dessen_ (maskulin/neutrum) und _deren_ (feminin/Plural) ersetzen den Besitzartikel. Das Nomen danach steht ohne Artikel: _…, dessen Auto…_, nicht _…, dessen das Auto…_. Auf B1 reicht es meist, diese Formen **zu erkennen**; aktiv produzieren musst du sie selten.

---

### 3.8 Infinitivsätze: um…zu, ohne…zu, anstatt…zu

Infinitivsätze haben kein eigenes Subjekt — das Subjekt ist dasselbe wie im Hauptsatz.

| Konstruktion | Bedeutung          | Beispiel                                                  |
| ------------ | ------------------ | --------------------------------------------------------- |
| um…zu        | Zweck / Ziel       | Ich lerne Deutsch, **um** in Deutschland **zu arbeiten**. |
| ohne…zu      | Ohne etwas zu tun  | Er ging weg, **ohne** sich **zu verabschieden**.          |
| anstatt…zu   | Statt etwas zu tun | Sie liest ein Buch, **anstatt** fern**zusehen**.          |

**Bei trennbaren Verben:** „zu" steht zwischen Präfix und Verb:

- _um ein**zu**kaufen_
- _ohne auf**zu**stehen_
- _anstatt mit**zu**kommen_

---

### 3.9 damit vs. um…zu

Beide drücken einen Zweck aus, aber:

|          | um…zu                                            | damit                                                  |
| -------- | ------------------------------------------------ | ------------------------------------------------------ |
| Subjekt  | Gleich im Haupt- und Nebensatz                   | Unterschiedliches Subjekt                              |
| Beispiel | **Ich** lerne Deutsch, um eine Arbeit zu finden. | **Ich** erkläre es langsam, damit **du** es verstehst. |
| Beispiel | **Er** spart Geld, um ein Auto zu kaufen.        | **Sie** spricht laut, damit **alle** sie hören.        |

> Wenn das Subjekt gleich ist, kann man beides verwenden, aber _um…zu_ ist eleganter.

---

### 3.10 Indirekte Rede (Grundlagen) — B1

Indirekte Rede gibt wieder, was jemand gesagt hat. Auf B1-Niveau verwendet man meistens _dass_-Sätze.

**Direkte Rede → Indirekte Rede:**

- _Er sagt: „Ich bin müde."_ → _Er sagt, **dass** er müde **ist**._
- _Sie fragt: „Kommst du mit?"_ → _Sie fragt, **ob** ich mitkomme._
- _Er hat gesagt: „Ich habe keine Zeit."_ → _Er hat gesagt, **dass** er keine Zeit **hat**._

**Ohne „dass" (mit normaler Hauptsatzstellung):**

- _Er sagt, er **ist** müde._

> **Konjunktiv I — nur erkennen (B1+):** In Nachrichten, Berichten und formellen Texten erscheint die indirekte Rede oft im Konjunktiv I: _Er sagte, er **sei** müde. / Sie sagte, sie **habe** keine Zeit._ Auf B1 musst du das **verstehen**, aber nicht aktiv bilden — im eigenen Schreiben genügen _dass_-Sätze im Indikativ.

---

### 3.11 je…desto / je…umso — B1

Diese Konstruktion drückt eine proportionale Beziehung aus: wenn A mehr wird, wird auch B mehr.

**Struktur:** je + Komparativ + … + Verb (am Ende), desto/umso + Komparativ + Verb + Subjekt

**Beispielsätze:**

- **\*Je** mehr ich lerne, **desto** besser verstehe ich.\*
- **\*Je** älter man wird, **desto** weiser wird man.\*
- **\*Je** früher du kommst, **umso** besser.\*
- **\*Je** länger man wartet, **desto** schwieriger wird es.\*

**Achtung:** Nach _desto/umso_ kommt Inversion (Verb vor Subjekt).

---

## 4. Fälle in Aktion

### 4.1 Nominativ — Das Subjekt

Der Nominativ antwortet auf die Frage **Wer?** oder **Was?** und markiert das Subjekt des Satzes.

**Beispielsätze:**

- **\*Der Hund** schläft.\*
- **\*Meine Mutter** kocht das Essen.\*
- **\*Das** ist **ein** Problem.\*
- **\*Wer** ist das?\* — Das ist **mein** Bruder.

Auch nach _sein_, _werden_, _bleiben_:

- _Er ist **ein guter Lehrer**._
- _Sie wird **Ärztin**._

---

### 4.2 Akkusativ — Direktes Objekt & Präpositionen

Der Akkusativ antwortet auf die Frage **Wen?** oder **Was?**

**Mit Verben (direktes Objekt):**

- _Ich kaufe **einen Kuchen**._
- _Sie liest **das Buch**._
- _Hast du **meinen Schlüssel** gesehen?_

**Verben mit Akkusativ:** haben, brauchen, kaufen, lesen, sehen, finden, suchen, besuchen, trinken, essen, nehmen…

**Akkusativ-Präpositionen:** für, gegen, ohne, durch, um

| Präposition | Beispiel                               |
| ----------- | -------------------------------------- |
| für         | Das Geschenk ist **für meinen Vater**. |
| gegen       | Ich bin **gegen diesen Plan**.         |
| ohne        | Er geht **ohne seinen Freund**.        |
| durch       | Wir gehen **durch den Park**.          |
| um          | Wir laufen **um den See**.             |

---

### 4.3 Dativ — Indirektes Objekt & Präpositionen

Der Dativ antwortet auf die Frage **Wem?**

**Mit Verben (indirektes Objekt):**

- _Ich gebe **dem Kind** ein Eis._
- _Er hilft **seiner Mutter**._
- _Kannst du **mir** bitte helfen?_

**Verben mit Dativ:** helfen, danken, gehören, gefallen, schmecken, fehlen, passen, antworten, gratulieren…

**Dativ-Präpositionen:** ab, aus, bei, mit, nach, seit, von, zu, gegenüber

| Präposition | Beispiel                                |
| ----------- | --------------------------------------- |
| ab          | **Ab nächster Woche** habe ich Urlaub.  |
| aus         | Sie kommt **aus der Türkei**.           |
| bei         | Er wohnt **bei seinen Eltern**.         |
| mit         | Ich fahre **mit dem Bus**.              |
| nach        | Nach **dem Essen** gehen wir spazieren. |
| seit        | Ich lerne **seit einem Jahr** Deutsch.  |
| von         | Das Buch ist **von meinem Bruder**.     |
| zu          | Ich gehe **zum Arzt**. (zu + dem = zum) |
| gegenüber   | Die Bank ist **gegenüber dem Bahnhof**. |

---

### 4.4 Wechselpräpositionen (Akkusativ oder Dativ)

Neun Präpositionen können sowohl mit Akkusativ als auch mit Dativ stehen:

**in, an, auf, über, unter, vor, hinter, neben, zwischen**

| Frage                 | Fall      | Bedeutung | Beispiel                        |
| --------------------- | --------- | --------- | ------------------------------- |
| **Wohin?** (Bewegung) | Akkusativ | Richtung  | Ich gehe **in den Park**.       |
| **Wo?** (Position)    | Dativ     | Ort       | Ich bin **im Park**. (in + dem) |

**Beispielpaare:**

| Wohin? (Akk.)                           | Wo? (Dat.)                         |
| --------------------------------------- | ---------------------------------- |
| Ich stelle das Buch **auf den Tisch**.  | Das Buch steht **auf dem Tisch**.  |
| Er hängt das Bild **an die Wand**.      | Das Bild hängt **an der Wand**.    |
| Sie legt das Kind **ins Bett**.         | Das Kind liegt **im Bett**.        |
| Ich setze mich **neben meinen Freund**. | Ich sitze **neben meinem Freund**. |

**Merkhilfe — Verben:**

- Bewegung (Akk.): stellen, legen, hängen (transitiv), setzen
- Position (Dat.): stehen, liegen, hängen (intransitiv), sitzen

---

### 4.5 Genitiv — B1

Der Genitiv drückt Besitz aus und steht nach bestimmten Präpositionen.

**Besitz:**

- _Das ist das Auto **meines Vaters**._ (maskulin: +s/es)
- _Die Tasche **der Frau** ist rot._ (feminin: keine Änderung am Nomen)
- _Die Spielsachen **des Kindes** liegen überall._ (neutrum: +s/es)
- _Die Meinungen **der Leute** sind verschieden._ (Plural)

**Genitiv-Präpositionen:**

| Präposition | Bedeutung           | Beispiel                                                 |
| ----------- | ------------------- | -------------------------------------------------------- |
| wegen       | Grund / Ursache     | **Wegen des Wetters** bleiben wir zu Hause.              |
| trotz       | Gegensatz / dennoch | **Trotz der Kälte** geht er joggen.                      |
| während     | Zeitraum            | **Während des Unterrichts** darf man nicht telefonieren. |
| (an)statt   | anstelle von        | **(An)statt des Kuchens** nehme ich Obst.                |

---

## 5. Adjektivdeklination

### 5.1 Nach bestimmtem Artikel (schwache Deklination)

Nach _der, die, das, diese, jede, alle_:

| Fall | Maskulin             | Feminin            | Neutrum              | Plural                |
| ---- | -------------------- | ------------------ | -------------------- | --------------------- |
| Nom. | der alt**e** Mann    | die alt**e** Frau  | das alt**e** Haus    | die alt**en** Häuser  |
| Akk. | den alt**en** Mann   | die alt**e** Frau  | das alt**e** Haus    | die alt**en** Häuser  |
| Dat. | dem alt**en** Mann   | der alt**en** Frau | dem alt**en** Haus   | den alt**en** Häusern |
| Gen. | des alt**en** Mannes | der alt**en** Frau | des alt**en** Hauses | der alt**en** Häuser  |

**Faustregel:** Meist **-en**, außer Nominativ (m/f/n) und Akkusativ (f/n) → **-e**.

---

### 5.2 Nach unbestimmtem Artikel (gemischte Deklination)

Nach _ein, kein, mein, dein, sein, ihr, unser, euer_:

| Fall | Maskulin               | Feminin              | Neutrum                | Plural (kein/mein…)      |
| ---- | ---------------------- | -------------------- | ---------------------- | ------------------------ |
| Nom. | ein alt**er** Mann     | eine alt**e** Frau   | ein alt**es** Haus     | keine alt**en** Häuser   |
| Akk. | einen alt**en** Mann   | eine alt**e** Frau   | ein alt**es** Haus     | keine alt**en** Häuser   |
| Dat. | einem alt**en** Mann   | einer alt**en** Frau | einem alt**en** Haus   | keinen alt**en** Häusern |
| Gen. | eines alt**en** Mannes | einer alt**en** Frau | eines alt**en** Hauses | keiner alt**en** Häuser  |

---

### 5.3 Ohne Artikel (starke Deklination)

Wenn kein Artikel vorhanden ist, übernimmt das Adjektiv die Endung des bestimmten Artikels:

| Fall | Maskulin           | Feminin            | Neutrum            | Plural                |
| ---- | ------------------ | ------------------ | ------------------ | --------------------- |
| Nom. | kalt**er** Kaffee  | frisch**e** Milch  | kalt**es** Wasser  | frisch**e** Brötchen  |
| Akk. | kalt**en** Kaffee  | frisch**e** Milch  | kalt**es** Wasser  | frisch**e** Brötchen  |
| Dat. | kalt**em** Kaffee  | frisch**er** Milch | kalt**em** Wasser  | frisch**en** Brötchen |
| Gen. | kalt**en** Kaffees | frisch**er** Milch | kalt**en** Wassers | frisch**er** Brötchen |

> **Hinweis zum Genitiv (m/n):** Weil das Nomen schon die Endung -s/-es trägt (_Kaffees_, _Wassers_), bekommt das Adjektiv hier ausnahmsweise **-en** statt -es. Das ist eine häufige Fehlerquelle.

**Beispielsätze:**

- _Ich trinke **kalten** Kaffee._ (ohne Artikel, Akk., maskulin)
- _Er isst gern **frische** Brötchen._ (ohne Artikel, Akk., Plural)

---

### 5.4 Komparativ & Superlativ

| Grundform | Komparativ (+er) | Superlativ (am …sten / der …ste) |
| --------- | ---------------- | -------------------------------- |
| schnell   | schneller        | am schnellsten / der schnellste  |
| groß      | größer           | am größten / der größte          |
| gut       | besser           | am besten / der beste            |
| viel      | mehr             | am meisten / der meiste          |
| gern      | lieber           | am liebsten                      |
| hoch      | höher            | am höchsten                      |
| alt       | älter            | am ältesten / der älteste        |
| jung      | jünger           | am jüngsten / der jüngste        |
| kalt      | kälter           | am kältesten / der kälteste      |
| kurz      | kürzer           | am kürzesten / der kürzeste      |
| nah       | näher            | am nächsten / der nächste        |

**Vergleiche:**

- Komparativ + **als**: _Er ist größer **als** ich._
- Gleichheit: **so…wie**: _Sie ist **so** alt **wie** ich._
- Superlativ: _Das ist **der beste** Film._

**Beispielsätze:**

- _Berlin ist **größer als** München._
- _Dieses Restaurant ist **so gut wie** das andere._
- _Das war **der schönste** Tag meines Lebens._
- _Ich lese **lieber** Bücher **als** Zeitungen._

**Attributive Komparative/Superlative (mit Adjektivendung):**

Komparative und Superlative vor einem Nomen bekommen zusätzlich die normale Adjektivendung:

- _ein **älterer** Mann_ (Komparativ + gemischte Endung -er)
- _die **bessere** Lösung_ (Komparativ + schwache Endung -e)
- _das **schönste** Haus_ (Superlativ + schwache Endung -e)
- _mit **größerem** Aufwand_ (Komparativ + starke Endung -em)

| ------------ | --------- | --------- | ----- |
| ich          | ich       | mich      | mir   |
| du           | du        | dich      | dir   |
| er           | er        | ihn       | ihm   |
| sie (f)      | sie       | sie       | ihr   |
| es           | es        | es        | ihm   |

| sie (Pl.)    | sie       | sie       | ihnen |
| Sie (formal) | Sie       | Sie       | Ihnen |

**Beispielsätze:**

- _Kannst du **mir** helfen?_ (Dativ)
- _Ich gebe **ihr** das Buch._ (Dativ)
- _Er hat **uns** eingeladen._ (Akkusativ)

### 6.2 Possessivpronomen

| Person       | Possessiv | Beispiel (Nom., m)        | Beispiel (Akk., m)             |
| ------------ | --------- | ------------------------- | ------------------------------ |
| ich          | mein      | **Mein** Hund ist groß.   | Ich liebe **meinen** Hund.     |
| du           | dein      | **Dein** Auto ist neu.    | Ich sehe **deinen** Bruder.    |
| er/es        | sein      | **Sein** Buch liegt hier. | Er sucht **seinen** Schlüssel. |
| sie (f)      | ihr       | **Ihr** Kind schläft.     | Sie liebt **ihren** Mann.      |
| wir          | unser     | **Unser** Haus ist klein. | Wir verkaufen **unser** Haus.  |
| ihr          | euer      | **Euer** Lehrer ist nett. | Kennt ihr **euren** Nachbarn?  |
| sie (Pl.)    | ihr       | **Ihr** Garten ist schön. | Sie pflegen **ihren** Garten.  |
| Sie (formal) | Ihr       | **Ihr** Name bitte?       | Ich brauche **Ihren** Pass.    |

> Possessivpronomen werden wie _ein/kein_ dekliniert.

### 6.3 Reflexivpronomen

(Siehe auch 2.8 — Reflexive Verben)

| Person    | Akkusativ | Dativ |
| --------- | --------- | ----- |
| ich       | mich      | mir   |
| du        | dich      | dir   |
| er/sie/es | sich      | sich  |
| wir       | uns       | uns   |
| ihr       | euch      | euch  |
| sie/Sie   | sich      | sich  |

**Häufige reflexive Verben:**

- sich freuen (auf/über), sich ärgern (über), sich erinnern (an)
- sich waschen, sich anziehen, sich setzen, sich fühlen
- sich entschuldigen, sich vorstellen, sich bewerben (um)

**Beispielsätze:**

- _Ich erinnere **mich** an den Urlaub._
- _Er bewirbt **sich** um die Stelle._
- _Wir freuen **uns** auf das Wochenende._

---

### 6.4 Relativpronomen — B1

(Siehe auch 3.7)

Die Relativpronomen sehen fast wie die bestimmten Artikel aus:

| Fall | Maskulin   | Feminin   | Neutrum    | Plural    |
| ---- | ---------- | --------- | ---------- | --------- |
| Nom. | der        | die       | das        | die       |
| Akk. | den        | die       | das        | die       |
| Dat. | dem        | der       | dem        | **denen** |
| Gen. | **dessen** | **deren** | **dessen** | **deren** |

**Achtung:**

- Im Dativ Plural ist das Relativpronomen **denen** (nicht „den").
- Im Genitiv weichen die Formen vom Artikel ab: **dessen** (m/n) und **deren** (f/Pl.). Das folgende Nomen steht **ohne Artikel**.

**Beispielsätze:**

- _Die Stadt, **in der** ich wohne, ist sehr schön._ (Dat., feminin)
- _Der Mann, **mit dem** sie arbeitet, ist Ingenieur._ (Dat., maskulin)
- _Die Leute, **denen** ich geholfen habe, waren dankbar._ (Dat., Plural)
- _Der Kollege, **dessen** Büro neben meinem ist, ist krank._ (Gen., maskulin)
- _Das Kind, **dessen** Mutter Lehrerin ist, sitzt vorne._ (Gen., neutrum)

---

### 6.5 Indefinitpronomen

| Pronomen | Bedeutung              | Beispiel                             |
| -------- | ---------------------- | ------------------------------------ |
| man      | „jeder" (allgemein)    | **Man** spricht hier Deutsch.        |
| jemand   | irgendeine Person      | Hat **jemand** meinen Stift gesehen? |
| niemand  | keine Person           | **Niemand** hat geantwortet.         |
| etwas    | eine unbestimmte Sache | Ich möchte **etwas** essen.          |
| nichts   | keine Sache            | Er weiß **nichts** davon.            |
| alle     | jeder (Plural)         | **Alle** sind eingeladen.            |
| jeder    | jede einzelne Person   | **Jeder** muss das verstehen.        |

**Beispielsätze:**

- **\*Man** darf hier nicht rauchen.\* (= Es ist verboten.)
- _Ich habe **niemanden** gesehen._
- _Gibt es noch **etwas** zu essen?_

## 7. Präpositionen

### 7.1 Feste Akkusativ-Präpositionen

**Merksatz:** **F-U-G-O-D** (für, um, gegen, ohne, durch)

| Präposition | Bedeutung | Beispiel                         |
| ----------- | --------- | -------------------------------- |
| für         | für       | Das ist **für dich**.            |
| um          | um, herum | Wir sitzen **um den Tisch**.     |
| gegen       | gegen     | Er ist **gegen die Idee**.       |
| ohne        | ohne      | Ich gehe **ohne meinen Bruder**. |
| durch       | durch     | Wir gehen **durch den Wald**.    |

---

### 7.2 Feste Dativ-Präpositionen

**Merksatz:** **Ab-aus-bei-mit-nach-seit-von-zu-gegenüber**

| Präposition | Bedeutung               | Beispiel                                  |
| ----------- | ----------------------- | ----------------------------------------- |
| ab          | ab (Zeitpunkt)          | **Ab morgen** mache ich Diät.             |
| aus         | aus (Herkunft/Material) | Sie kommt **aus dem Iran**.               |
| bei         | bei                     | Er arbeitet **bei Siemens**.              |
| mit         | mit                     | Ich fahre **mit dem Zug**.                |
| nach        | nach (Richtung/Zeit)    | **Nach dem Kurs** gehe ich einkaufen.     |
| seit        | seit (Zeitdauer)        | Ich wohne **seit drei Jahren** hier.      |
| von         | von                     | Das Geschenk ist **von meiner Freundin**. |
| zu          | zu (Richtung)           | Ich gehe **zum Supermarkt**.              |
| gegenüber   | gegenüber               | Die Bank ist **gegenüber dem Bahnhof**.   |

**Kurzformen (Verschmelzungen):**

- bei + dem = beim
- von + dem = vom
- zu + dem = zum
- zu + der = zur

---

### 7.3 Wechselpräpositionen

(Siehe auch 4.4)

**in, an, auf, über, unter, vor, hinter, neben, zwischen**

**Kurzformen:**

- in + dem = im, in + das = ins
- an + dem = am, an + das = ans

| Präposition | Wo? (Dativ)                         | Wohin? (Akkusativ)                    |
| ----------- | ----------------------------------- | ------------------------------------- |
| in          | Ich bin **im** Kino.                | Ich gehe **ins** Kino.                |
| an          | Das Bild hängt **am** Fenster.      | Ich hänge es **ans** Fenster.         |
| auf         | Das Glas steht **auf dem** Tisch.   | Ich stelle es **auf den** Tisch.      |
| über        | Die Lampe hängt **über dem** Tisch. | Häng die Lampe **über den** Tisch.    |
| unter       | Die Katze ist **unter dem** Bett.   | Die Katze kriecht **unter das** Bett. |

---

### 7.4 Genitiv-Präpositionen — B1

| Präposition | Bedeutung    | Beispiel                                         |
| ----------- | ------------ | ------------------------------------------------ |
| wegen       | Grund        | **Wegen des Staus** komme ich zu spät.           |
| trotz       | trotz        | **Trotz des Regens** gehen wir spazieren.        |
| während     | zeitlich     | **Während der Prüfung** darf man nicht sprechen. |
| (an)statt   | anstelle von | **(An)statt des Kuchens** nehme ich Obst.        |

---

### 7.5 Temporale Präpositionen: seit vs. vor, am/um/im

| Situation                      | Präposition          | Beispiel                                 |
| ------------------------------ | -------------------- | ---------------------------------------- |
| Dauer bis jetzt (Präsens!)     | **seit** + Dativ     | Ich lerne **seit zwei Monaten** Deutsch. |
| Zeitpunkt in der Vergangenheit | **vor** + Dativ      | **Vor drei Tagen** war ich beim Arzt.    |
| Tage                           | **am**               | **Am** Montag habe ich frei.             |
| Uhrzeiten                      | **um**               | Der Kurs beginnt **um** 9 Uhr.           |
| Monate / Jahreszeiten          | **im**               | **Im** Juli fahre ich in den Urlaub.     |
| Jahre                          | **—** (kein Artikel) | **2024** habe ich die Prüfung gemacht.   |

**Wichtig — seit + Präsens:**

- _Ich wohne **seit** 2020 in Berlin._ (NICHT: „Ich habe gewohnt")
- _Er arbeitet **seit** einem Jahr hier._

---

### 7.6 Verb-Präposition-Verbindungen

Viele Verben haben eine feste Präposition. Man muss sie auswendig lernen.

| Verb + Präposition     | Fall | Beispiel                              |
| ---------------------- | ---- | ------------------------------------- |
| warten auf             | Akk. | Ich warte **auf den Bus**.            |
| sich freuen auf        | Akk. | Ich freue mich **auf den Urlaub**.    |

## 8. Negation & Wortbildung

### 8.1 nicht vs. kein

|                    | kein                                               | nicht                                                      |
| ------------------ | -------------------------------------------------- | ---------------------------------------------------------- |
| Was wird verneint? | Nomen (mit unbestimmtem Artikel oder ohne Artikel) | Verben, Adjektive, Adverbien, Nomen mit bestimmtem Artikel |
| Beispiel           | Ich habe **keinen** Hund.                          | Ich komme heute **nicht**.                                 |

**kein verwenden:**

- _Ich habe **kein** Auto._ (statt: ein Auto)
- _Er trinkt **keinen** Kaffee._ (statt: einen Kaffee)
- _Wir haben **keine** Zeit._ (statt: Zeit — ohne Artikel)
- _Das ist **kein** Problem._

**nicht verwenden:**

- _Ich komme **nicht**._ (Verb verneinen)
- _Das Essen ist **nicht** gut._ (Adjektiv verneinen)
- _Er arbeitet **nicht** gern._ (Adverb verneinen)
- _Das ist **nicht** der richtige Weg._ (bestimmter Artikel)
- _Ich kenne ihn **nicht**._ (Pronomen/Akkusativobjekt)

**Position von „nicht":**

- Am Satzende (wenn kein Satzende-Element da ist): _Ich verstehe das **nicht**._
- Vor Adjektiven/Adverbien: _Das ist **nicht** schön._
- Vor Präpositionalphrasen: _Ich gehe **nicht** ins Kino._
- Vor dem zweiten Verbteil: _Er hat mich **nicht** angerufen._

---

### 8.2 Zusammengesetzte Nomen (Komposita)

Im Deutschen kann man Nomen zusammensetzen. Das **letzte Wort** bestimmt den Artikel.

| Wort 1     | + Wort 2      | = Kompositum       | Artikel         |
| ---------- | ------------- | ------------------ | --------------- |
| die Hand   | + das Tuch    | = das Handtuch     | das (← Tuch)    |
| der Schlaf | + das Zimmer  | = das Schlafzimmer | das (← Zimmer)  |
| die Geburt | + der Tag     | = der Geburtstag   | der (← Tag)     |
| das Haus   | + die Aufgabe | = die Hausaufgabe  | die (← Aufgabe) |

**Manchmal mit Fugen-s oder Fugen-n:**

- Arbeit**s**platz, Geburt**s**tag, Lieblings**buch**, Meinung**s**freiheit
- Straße**n**bahn, Sonne**n**brille

**Beispielsätze:**

- _Der **Kühlschrank** ist leer._ (kühl + Schrank)
- _Ich mache meine **Hausaufgaben**._ (Haus + Aufgaben)
- _Hast du deinen **Reisepass**?_ (Reise + Pass)

### 8.3 Häufige Vor- und Nachsilben

**Vorsilben (Präfixe) — verändern die Bedeutung:**

| Präfix | Bedeutung            | Beispiele                                       |
| ------ | -------------------- | ----------------------------------------------- |
| un-    | Negation / Gegenteil | unfreundlich, unmöglich, unglücklich, unhöflich |
| miss-  | falsch / schlecht    | missverstehen, missbrauchen, Misserfolg         |

**Nachsilben (Suffixe) — helfen das Genus zu erkennen:**

| Suffix                | Genus         | Beispiele                                                               |
| --------------------- | ------------- | ----------------------------------------------------------------------- |
| -ung                  | feminin       | die Wohnung, die Übung, die Entscheidung                                |
| -heit                 | feminin       | die Freiheit, die Gesundheit, die Schönheit                             |
| -keit                 | feminin       | die Möglichkeit, die Schwierigkeit                                      |
| -tion                 | feminin       | die Information, die Situation                                          |
| -schaft               | feminin       | die Freundschaft, die Mannschaft                                        |
| -er (Personen/Geräte) | maskulin      | der Lehrer, der Computer                                                |
| -ling                 | maskulin      | der Frühling, der Lehrling                                              |
| -chen / -lein         | neutrum       | das Mädchen, das Brötchen                                               |
| -ment                 | meist neutrum | das Dokument, das Instrument (aber: **der** Zement, **der** Konsument)  |
| -nis                  | meist neutrum | das Ergebnis, das Geheimnis (aber: **die** Erlaubnis, **die** Kenntnis) |

## 9. Register & Redemittel (TELC)

### 9.1 Formell (Sie) vs. Informell (du)

Im Deutschen unterscheidet man klar zwischen formeller und informeller Sprache. In der TELC-Prüfung muss man das Register konsequent durchhalten.

|               | Informell (du)             | Formell (Sie)                                         |
| ------------- | -------------------------- | ----------------------------------------------------- |
| Anrede        | du / ihr                   | Sie                                                   |
| Possessiv     | dein / euer                | Ihr                                                   |
| Verb          | Kommst du...?              | Kommen Sie...?                                        |
| Imperativ     | Ruf mich an!               | Rufen Sie mich an!                                    |
| Brief-Anrede  | Lieber Thomas / Liebe Anna | Sehr geehrter Herr Müller / Sehr geehrte Frau Schmidt |
| Brief-Schluss | Viele Grüße / Liebe Grüße  | Mit freundlichen Grüßen                               |

**Wann formell?**

- Behörden, Ämter, unbekannte Personen, Geschäftsbriefe, Beschwerden

**Wann informell?**

- Freunde, Familie, enge Kollegen (wenn man sich duzt)

### 9.3 Meinung ausdrücken

| Funktion      | Redemittel                                                                                                  |
| ------------- | ----------------------------------------------------------------------------------------------------------- |
| Meinung sagen | Ich bin der Meinung, dass… / Meiner Meinung nach… / Ich finde, dass… / Ich denke, dass… / Ich glaube, dass… |
| Zustimmen     | Da hast du recht. / Ich bin (ganz) deiner/Ihrer Meinung. / Das stimmt. / Genau!                             |
| Widersprechen | Da bin ich anderer Meinung. / Das sehe ich anders. / Ich bin nicht einverstanden. / Das stimmt nicht ganz.  |

**Beispielsätze:**

- **\*Meiner Meinung nach** sollte man mehr Sport treiben.\*
- _Ich finde, **dass** öffentliche Verkehrsmittel wichtig **sind**._
- **\*Da bin ich anderer Meinung.** Ich denke, dass…\*

### 9.4 Höfliche Bitten & Vorschläge

| Funktion          | Redemittel                                                                                      |
| ----------------- | ----------------------------------------------------------------------------------------------- |
| Bitte (formell)   | Könnten Sie mir bitte…? / Wären Sie so freundlich…? / Dürfte ich Sie bitten…? / Ich hätte gern… |
| Bitte (informell) | Kannst du mir…? / Würdest du bitte…? / Hättest du vielleicht…?                                  |
| Vorschlag         | Ich schlage vor, dass… / Wie wäre es, wenn…? / Wir könnten… / Sollen wir…?                      |
| Einladung         | Ich möchte dich/Sie einladen. / Hast du Lust…? / Hättest du Zeit…?                              |
| Absage            | Leider kann ich nicht. / Es tut mir leid, aber… / Ich würde gern, aber…                         |

**Beispielsätze:**

- **\*Könnten Sie** mir bitte sagen, wann der Kurs beginnt?\*
- **\*Wie wäre es**, wenn wir am Samstag zusammen kochen?\*
- _Es tut mir leid, aber ich **kann** leider **nicht** kommen._
- _Ich **hätte** gern einen Termin am Dienstag._

### 9.5 Textverbindung & Struktur

Für zusammenhängende Texte in der Prüfung:

| Funktion        | Wörter/Phrasen                                                          |
| --------------- | ----------------------------------------------------------------------- |
| Reihenfolge     | Zuerst… / Dann… / Danach… / Anschließend… / Zum Schluss… / Schließlich… |
| Grund           | weil / da / denn / nämlich / deshalb / deswegen / aus diesem Grund      |
| Gegensatz       | aber / trotzdem / allerdings / obwohl / jedoch                          |
| Ergänzung       | außerdem / zusätzlich / darüber hinaus / auch / nicht nur…sondern auch  |
| Beispiel        | zum Beispiel / beispielsweise / wie                                     |
| Zusammenfassung | Zusammenfassend kann man sagen, dass… / Insgesamt… / Alles in allem…    |

**Beispiel — Strukturierter Text:**

> _Meiner Meinung nach ist Sport sehr wichtig. **Erstens** hält er den Körper gesund. **Zweitens** fühlt man sich danach besser. **Außerdem** kann man neue Leute kennenlernen. **Allerdings** hat nicht jeder genug Zeit dafür. **Trotzdem** sollte man versuchen, mindestens zweimal pro Woche Sport zu treiben. **Zusammenfassend** kann man sagen, dass Bewegung ein wichtiger Teil des Lebens ist._
